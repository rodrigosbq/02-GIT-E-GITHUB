Entendendo sobre Branches

Branches são ramificações do código Master ou Main que podem trabalhar de forma independentes do fluxo principal do código.
Em geral as Branches são utilizadas para que outros programadores possam trabalhar no mesmo código sem interferir no fluxo principal do projeto,
Seja para fazer testes, correções de bugs ou desenvolver novas aplicações, é normal o uso de Branches para os trabalhos em equipe.

Para iniciar uma nova Branch procurar pelo nome master ou main, no canto inferior esquerdo do VSCODE e clicar em Create New Branch.
Em geral no mercado de trabalho é criada uma Branch chamada "develop" para que o projeto possa ser testado sem alterar a estrutura que está em funcionamento do código.
Importante: Para se trocar de Branch é necessário que seja realizado o comit das mudanças no codigo com o intuito de resguardar a segurança do que foi realizado.

Pode ser realizado ainda uma Branch de uma Branch, neste caso deve ser clicado no botão "+ Create new branch from", escolher o nome e depois selecionar de qual Branch os dados serão "duplicados"
