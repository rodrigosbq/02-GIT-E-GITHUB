Criando o Primeiro Repositorio


1- Procurar o icone com 3 bolinhas com o nome Controle de Código-GFonte ( Source Control), ou acessar pelo atalho crtl+shitf+G.
2- CLicar em initialize repository
3- Os arquivos deverão estar com um símbolo "U" ao lado do seu nome
o que quer dizer que eles ainda não estão com seu histórico de modificações monitorados
4- No controle de código fonte, ao lado de cada arquivo que desejar guardar o historico do arquivo clicar no botão "+".
5- O arquivo passará da lista Changes para a lista Staged Changes. Ou seja, estão separados para realizar o controle de versão.
6- Realizando um Commit , Deve ser escrita uma mensagem no Commit e os arquivos devem estar na lista Staged Changes.

SS- Extensões : Git Graph, na barra de baixo aparece o nome Gitgraph com os detalhamentos do commit.
Gitlens, mostra dentro do código , linha a linha clicada quem e quando foi realizada as alterações e habilita outras funções dentro do controle de código.
