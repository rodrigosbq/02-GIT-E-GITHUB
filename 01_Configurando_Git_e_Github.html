Sobre Git e Github

Git é o programa que ficará instalado em seu computador e realizará o controle de versão do código.
Para instalá-lo basta procurar por Git no Google ou ir no site https://git-scm.com/

Github é a página da internet onde podemos fazer o upload dos nossos códigos e controles de versão
Meu Github é https://github.com/rodrigosbq

Para confirgurar o github no seu computador/VsCode seguir os passos:
1- abrir o VSCODE
2- Clicar em terminal/novo terminal
3- Procurar pelo ícone "+", selecionar Git Bash e depois clicar novamente no "+" e selecionar perfil padrão Git Bash,
outra forma de selecionar o perfil padrão é ctrl+shift+p e digitar select default profile

4- No bash inserir as cofigurações:
git config --global user.email seu-email
git config --global user.name seu-username
git config --list ( para conferir se seus dados estão linkados)

5- Clicar no perfil de contas do VScode, logar com github, ativar vinculo no site do git hub e conferir no seu perfil se aparece seu username(github)
