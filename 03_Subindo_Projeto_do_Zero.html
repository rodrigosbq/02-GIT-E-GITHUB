SUBINDO PROJETO DO ZERO INICIANDO PELO GIT HUB

01- No seu github procure pelo campo repositories->new.
02- Coloque um nome para o repositorio, defina se quer que o mesmo seja público ou privado e clique em create repository.
03- No seu VSCodde , via terminal digite:

  • git init
  • git add .
  • git commit -m"nome do commit"
  • No repositorio criado pelo git, pegar a linha que starta com git remote add origin
    Ex: remote add origin https://github.com/rodrigosbq/01-Aprendendo-sobre-HTML.git
  • git push
04- Lembrar que você deve estar logado no VSCODE.
